Aprende JavaScript.

JavaScript es un lenguaje de programación ligero, interpretado, con funciones de primera clase Y tipado debil.

Si bien es más conocido como un lenguaje de scripting para páginas web, es usado en muchos entornos fuera del navegador.

JavaScript Llego para quedarse, aprende mas en esta sitio Web.

Variables en JavaScript

let, var y const

Son palabras reservadas para definir variables,

  • let es una variable de scope local.
  • var es una variable en la cual su scope puede ser abstraido a todo el programa.
  • const se utiliza para almacenar constantes.

Todas estas variables pueden ser de cualquier tipo de dato, caracteres, arreglos, numeros, objetos e incluso funciones.

Objetos en JavaScript

Objetos como estructura de datos

Son un conjunto de datos ordenados en forma de lista con un indice definido por nosotros.

Estos pueden albergar cualquier tipo de dato, caracteres, arreglos, numeros, objetos e incluso funciones.

Condicionales en JavaScript

if, else, switch

Son estructuras de control que nos permiten seleccionar un trozo de codigo.

Estos por supuesto varian dependiendo de la interaccion de la entrada.

Bucles en JavaScript

for, while, do-while

Son estructuras de control reutilizar codigo repitiendolo siempre y cuando se cumpla una condicion

Sumamente efectivo para ahorrar codigo de forma eficiente o recorrer estructuras de datos.

Destructuring en JavaScript

Propiedades de objetos

El destructuring es una forma moderna que tenemos en JacaScript de acceder a las propiedades de un objeto.

Esto nos ahora codigo, esfuerzo, y por supuesto hace el codigo mas legible y estetico

DOM en JavaScript

Document Object Model

El DOM es la forma que tiene Javascript de ver todos los documentos que son capaces de ser reproducidos por un navegador.

JavaScript interpreta todos estis elementos como si fuesen objetos, esa es su manera de interactuar con todo.

JSON en JavaScript

JavaScript Object Notation

JSON es un tipo de archivo que utiliza JavaScript para interpretar toda la informacion de entrada.

Todo dato que reciba este lenguaje es procesado en tipo JSON

Arrow functions en JavaScript

Arrow function

Las Arrow function o "Lambdas" como se les conoce en otros lenguaje son funciones sin nombre.

Estas fueron creadas con el fin de ahorrar codigo, resolver problemas conplejos de un solo uso y ejecutar instrucciones sin parametro, ampliamente utilizada en el lado del servidor.

Async en JavaScript

Funciones Asincronas

Las sunciones asincronas son la esencia de JavaScript

Te permiten ejecutar codigo mientras se resuelve una peticion.

HTTP Request en JavaScript

AJAX HTTPRequest

El metodo HTTPRequest viene implementado en JavaScript de la mano de AJAX para consumir apis del lado del cliente.

Esto abrio todo un mundo nuevo de posibilidades ya que con el uso de librerias remotas se le saco partido a los frameworks de Javascript.

APIs en JavaScript

API

El metodo mas comun para establecer comunicaciones con un objeto remoto.

En JavaScript normalmente se reciben datos de la API mediante intercambio de archivos .JSON.

React en JavaScript

React.js

El framework de frontend con excelencia de Javacript.

Debido a su gran velocidad, confiabilidad y adaptabilidad se ha vuelto el favotiro de la comunidad, desarrollado por Facebook para volver a JavaScript un gigante la programacion moderna.

Angular en JavaScript

Angular.js

Creado y mantenido por Google.

Fue el primer framework de frontend de alto rendimiento que aparecio en el mercado, aunque actualmente se utilza con TypeScript.

Vue en JavaScript

Vue.js

Creado por Google y mantenido por la comunidad.

Quizas de estos tres es el framework de frontend menos popular pero el mas amado debido a lo facil que es para principiantes.

Node en JavaScript

Node.js

Creado por Ryan Dahl.

Node.js es la forma de extender Javascript fuera del navegador, dandonos a dia de hoy la posibilidad de usarlo para cualquier cosa, IoT, IA, aplicaciones moviles, Aplicaciones de escritorio, Backend e incluso bases de datos NO-Sql.
